Quick Start

# Compress a PowerPoint presentation
python src/deck_compress.py presentation.pptx
# Compress with custom quality settings
python src/deck_compress.py presentation.pptx -q 80 -w 1920
# Compress a video file
python src/deck_compress.py video.mp4 --video-crf 25
# Batch process all files in a folder
python src/deck_compress.py /path/to/folder --folder --min-size 100
# Show help for all options
python src/deck_compress.py --help

Advanced Usage

Command Line Options

python src/deck_compress.py [options] input_file
-q, --quality JPEG quality (1-100, default: 85)
-w, --max-width Maximum image width (default: 1920)
--video-crf Video quality (0-51, default: 28)
--folder Process all files in folder
--min-size Minimum file size in MB (default: 100)
-f, --force Overwrite existing files
