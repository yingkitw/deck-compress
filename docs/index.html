Installation

  • 1 Install Python 3.8 or higher
  • 2 Install FFmpeg for video compression
  • 3 Install Python dependencies
  • 4 Choose your interface: Web or CLI
    • # Install FFmpeg
    # macOS: brew install ffmpeg
    # Ubuntu: sudo apt install ffmpeg
    # Windows: Download from ffmpeg.org
    # Install Python dependencies
    pip install -r requirements.txt
    # Web Interface (Recommended)
    python web_app/run.py
    # Open http://localhost:8000
    # CLI Tool
    python src/deck_compress.py presentation.pptx --ultra

    How to Use

    # Web Interface (Easiest)
    python web_app/run.py
    # Open http://localhost:8000 and drag & drop files
    # CLI - Ultra compression (50-80% reduction)
    python src/deck_compress.py presentation.pptx --ultra
    # CLI - Standard compression
    python src/deck_compress.py slides.pptx
    # CLI - Batch processing
    python src/deck_compress.py /path/to/folder --folder --ultra
    # CLI - Image and video files
    python src/deck_compress.py image.png --ultra
    python src/deck_compress.py video.mp4 --video-crf 25
