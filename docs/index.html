    How to Use

    # Compress PowerPoint with image optimization
    python src/deck_compress.py presentation.pptx
    # Ultra-aggressive compression for maximum size reduction
    python src/deck_compress.py slides.pptx --ultra
    # Compress with custom image quality (JPEG/PNG optimization)
    python src/deck_compress.py slides.pptx -q 85 -w 1920
    # Compress Word document with media optimization
    python src/deck_compress.py document.docx
    # Compress video with quality control
    python src/deck_compress.py video.mp4 --video-crf 25
    # Batch process with image compression
    python src/deck_compress.py /path/to/folder --folder --min-size 50
    # Show all compression options
    python src/deck_compress.py --help

    Compression Options

    -q, --quality JPEG quality 1-100 (default: 85) - PNG uses optimization
    -w, --max-width Max image width in pixels (default: 1920) - Auto-resize large images
    --video-crf Video quality 0-51 (default: 28) - Lower = better quality
    --ultra Ultra-aggressive compression mode (smaller files, lower quality)
    --folder Process all supported files in folder
    --min-size Minimum file size in MB to process (default: 100)
    -f, --force Overwrite existing compressed files

    üí° Image Compression Tips

    • JPEG: Quality 85-95 for presentations, 70-85 for smaller files
    • PNG: Preserves transparency, optimized for graphics and logos
    • Ultra Mode: Use --ultra for maximum compression (50-80% size reduction)
    • Progressive JPEG: Better compression with progressive encoding
    • Auto-resize: Large images automatically resized to max-width
    • Format detection: Automatically handles JPEG, PNG, BMP, TIFF, GIF
